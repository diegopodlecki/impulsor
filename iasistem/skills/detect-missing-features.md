# Skill: detect-missing-features

## Objetivo
Detectar funcionalidades clave faltantes.

## Basado en:
- tipo de proyecto (memory)
- buenas prácticas modernas

## Detectar:
- falta de auth
- falta de manejo de errores
- falta de estados vacíos
- falta de loading states
- falta de SEO básico
- falta de analytics

## Output
- lista priorizada de features faltantes

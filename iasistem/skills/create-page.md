# Skill: create-page

## Objetivo
Crear una nueva página en Next.js App Router.

## Input
- nombre de la página
- objetivo (landing, dashboard, etc.)

## Output
- archivo en /app
- layout básico
- componentes iniciales

## Reglas
- usar Tailwind
- mobile-first
- estructura clara
Si existe error, iterar hasta resolverlo antes de finalizar.

# Skill: debug-app

## Objetivo
Detectar y corregir errores en el proyecto automáticamente.

## Proceso
1. Ejecutar build
2. Detectar errores:
   - compilación
   - imports
   - tipos
   - runtime
3. Aplicar fixes directos
4. Reintentar build

## Output
- errores detectados
- código corregido

## Reglas
- no pedir intervención manual
- no romper funcionalidades existentes
- priorizar solución mínima efectiva

Si existe error, iterar hasta resolverlo antes de finalizar.

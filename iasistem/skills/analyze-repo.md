# Skill: analyze-repo

## Objetivo
Entender la estructura completa del proyecto.

## Proceso
1. Analizar carpetas:
   - /app
   - /components
   - /lib
   - /prisma

2. Identificar:
   - funcionalidades existentes
   - rutas implementadas
   - componentes reutilizables
   - modelos de datos

## Output
- mapa del sistema
- lista de features actuales

# Skill: implement-spec

## Objetivo
Implementar una funcionalidad basada en una spec definida.

## Input
- spec (.md)

## Proceso
1. Leer spec completa
2. Identar:
   - estructura
   - reglas
   - restricciones
3. Generar código alineado

## Output
- implementación completa
- código listo para producción

## Reglas
- NO inventar funcionalidades fuera de la spec
- respetar stack definido
- asegurar que todo lo definido funcione

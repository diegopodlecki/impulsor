## Screenshots de demos

Generá y subí acá las capturas reales de cada demo con estos nombres:

- `personal-trainer-preview.jpg` 1440x900
- `psicologo-preview.jpg` 1440x900
- `gimnasio-preview.jpg` 1440x900
- `nutricionista-preview.jpg` 1440x900
- `casas-de-comidas-preview.jpg` 1440x900
- `estetica-corporal-preview.jpg` 1440x900

### Cómo sacarlas

1. Abrí cada demo en Chrome con ancho de 1440 px.
2. Usá la extensión `Full Page Screen Capture` o Chrome DevTools -> `Capture screenshot`.
3. Guardá cada archivo con el nombre indicado arriba.
4. Comprimí en https://squoosh.app hasta dejar cada imagen por debajo de 100 KB.

### Nota

Mientras no estén las imágenes reales, la galería usa un fallback visual para no romper el sitio.
